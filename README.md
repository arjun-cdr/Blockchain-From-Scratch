# Blockchain-From-Scratch-Python
<h3>A custom modular, decentralised Blockchain built step-by-step from scratch using Python. It implements cryptographic validation using hashing techniques like SHA-256, Proof of Work Mining, P2P consensus sync engine and Smart Contracts. </h3>
<h4> Step 1 : Transaction Ledger - Implements a basic transaction ledger where a new transaction is validated if and only if it matches validation criteria</h4>
